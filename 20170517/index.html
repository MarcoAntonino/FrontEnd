Usa diversi linguaggi ed è mobile first. Essendo mobile first usa il viewport (sempre usato da noi). C'è anche il normalize css è un file css molto stupido che carichi prima del tuo css e fa in modo che i vari browser non abbiano i loro stili. Bootstrap lo vai a sovrascrivere. Container tiene le dimensioni fisse. Container fluid si allarga all'allargarsi alla proporzione dell schermo. La cosa davvero bella di bs è la griglia: perchè con le classiche layout dati da bootsrap si possono sistemare i vari elementi flottanti. Bs ti da due classi: la prima row. La riga dove andrò a mettere i miei contenutori flottanti. Di solito ci sono 12 colonne e bs ragiona a 12 colonne. Quindi div class row con la proporzioni: magari 8 e 4 (8+4=12).
